### Descrição

Landing Page feita com REACT E SCSS 


### Clonar o repositório


```bash
$ git clone https://github.com/edsuuu/landing-page-nft.git
```


### Acessar a pasta do repositorio

```bash
$ cd landing-page-nft
```

### Instalar as Dependencias


```bash
$ npm install
```
### Iniciar o projeto Local

```bash
$ npm start
```
